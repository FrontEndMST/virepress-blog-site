# 小米前端实习生一面

## 面试：

1. 详细说一下项目中提到的自动化部署

2. 代码题：盒子长宽比4:3，然后文字居中

3. 你了解哪些动画？

4. JS动画通常是使用`setInterval`这类方法，还是什么别的函数，为什么？

5. 代码题：`rAF`实现一个`setTimeout`函数

6. 场景题：`Dom`元素与视窗交叉时展示动画，并且只在第一次展示动画（口撕）

7. 场景题：每个用户在每天第一次进入时弹出一个提示，后续都不弹出
